# insomnia-plugin-firebase

Retornará o uid ou o token do usuário do firebase.

## Setup

Adicione nas variaveis de ambiente o seguinte atributo:
```json
{
  "firebase": {
    "config": {
      "apiKey": "",
      "authDomain": "",
      "databaseURL": "",
      "projectId": "",
      "storageBucket": "",
      "messagingSenderId": "",
      "appId": "",
    }
  }
}
```
